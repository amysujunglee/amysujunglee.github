'Wow Tasty' Admin System Renewal

Overview

Wow Tasty is about food delivery&take-out online ordering system in Vancouver.
The company has to manage more than 50 restaurants, 10 delivery staff, and more than 100 orders every day. The admin system allows real-time orders to arrive at the order management team immediately and assigns orders to delivery staff in the admin system. Restaurant owners are able to change restaurant information and menu information whenever they wanted.

My Roles

  • Researched the latest technology of real-time and automation admin system
  • Analyzed As-Is admin system and found out the pain points and solutions
  • Created the page list of To-Be admin system for each user type; Guest, Member, Delivery staff, Restaurant member, Restaurant manager, Admin member, and Admin manager
  • Created the wireframes and prototypes
  • Lead front-end development using HTML, CSS, jQuery, and JavaScript
  • Conducted usability testing

Work Process

(1) Research

  • Admin system goal and purpose research
  • Admin system trend research
  • UX/UI trend research

(2) Analyze

  • As-Is admin system analysis
  • Stakeholder and delivery staff interview analysis to find out the pain points and solutions

(3) Design

  • Required functions list
  • Page lists for each user type
  • To-Be wireframes and prototypes

(4) Develop/Test

  • Web development
  • Usability testing

Result

  • All information is automatically refreshed in 5 to 10 seconds increments.
  • The basic form and functions of the dashboard are provided consistently, but the access rights and the layout for each user type are separately created.
  • Real-time new orders can be seen immediately in the top right of the admin system via dynamic alert design.
