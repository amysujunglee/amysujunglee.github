MiniBox Website Development

Overview

The site is about renting mobile storages in Canada. The header is fixed so that users can see the menus while scrolling the website. There is a request form to get a free quote on the main slide. In subpages, users are able to compare the size and specifications of the mobile storages, then they order what they need to rent through the website.

Work Process

(1) Interviewed the MiniBin's owner and analyzed the website's goal and target audience

(2) Benchmarked the competitors' website and web marketing; M&M Services, Bin There Dump That, Metro Vancouver Dumpster Rentals, Trashking, etc

(3) Conducted the keyword research and applied Google's keyword algorithm and strategies
