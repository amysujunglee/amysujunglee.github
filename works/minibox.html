MiniBox Website Development

Overview

The site is about renting mobile storages in Canada. The header is fixed so that users can see the menus while scrolling the website. There is a request form to get a free quote on the main slide. In sub pages, users are able to compare the size and sepcifications of the mobile storages, then they order what they need to rent through the website.
