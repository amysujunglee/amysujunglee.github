How to use Git & GitHub efficently

January 1, 2014 by Amy Lee

Git is a version control software. You can simply check the history of your project or files. When you and your team members work several coding projects, you will see the high efficiency in your team and project using the software, Git.

Then.. What is GitHub? GitHub is a web service to see the history of the project or files.

First of all, sign up GitHub at https://github.com/

(Mac Version)
1. Open 'Terminal'
2. Type 'cd desktop' = 'Open Desktop' (cd = open)
3. 'git init' = To create a folder in desktop
* 'ls' -> To see all the filss in the folder
4. 'git clone https://github.com/amysujunglee/dice-game.git' = To create my repository folder in the selected folder in desktop
5. 'brackets' = To open brackets (code editor)
6. Once you finish the coding, 'git status' -> There will be red texts for the added files.
7. 'git add filename or add .'
8. 'git commit -m "comments"
9. git push origin master - DONE

Watch this video to set up Git: https://www.youtube.com/watch?v=x0EYpi38Yp4&t=1061s
